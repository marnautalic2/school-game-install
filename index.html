iOS (App Store)

  1. Otvori App Store poveznicu.
  2. Dodirni Preuzmi ili Instaliraj.
  3. Pokreni aplikaciju nakon instalacije.

Aplikacija je distribuirana preko privatne (unlisted) App Store poveznice.

Android (APK, GitHub Releases)

  1. Otvori Android poveznicu i preuzmi APK iz GitHub Releases.
  2. Kad se datoteka preuzme, dodirni APK datoteku.
  3. Ako Android blokira instalaciju, uključi "Dopusti iz ovog izvora" za preglednik.
  4. Vrati se na APK i dodirni Instaliraj.

Android može prikazati upozorenje za nepoznat izvor. To je normalno za internu instalaciju.

Ne radi instalacija? Otvori kratke upute.
  • Najčešći put: Postavke -> Sigurnost/Privatnost -> Instaliraj nepoznate aplikacije.
  • Odaberi preglednik koji je preuzeo APK (npr. Chrome) i uključi "Dopusti iz ovog izvora".
  • Nakon instalacije možeš ponovno isključiti tu opciju.
