iOS (TestFlight)

  1. Install the TestFlight app from Apple.
  2. Open the TestFlight public link.
  3. Tap Install inside TestFlight.

First external build needs a short TestFlight beta review.

Android (APK)

  1. Open the APK link in your browser.
  2. Allow install from this source if prompted.
  3. Tap Install when the download finishes.

Android will warn about unknown sources. This is normal for internal apps.
